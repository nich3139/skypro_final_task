Команды Git

  • git init - инициализация нового репозитория Git в текущей папке
  • git clone [url] - клонирование удаленного репозитория по указанному URL
  • git add [файл] - добавление файла в индекc для последующего коммита
  • git commit -m "сообщение" - создание коммита с указанным сообщением
  • git status - просмотр статуса изменений в рабочей директории и индексе
  • git push - отправка коммитов в удаленный репозиторий
  • git pull - получение изменений из удаленного репозитория и слияние с текущей веткой
  • git branch - просмотр, создание и удаление веток
  • git checkout [ветка] - переключение на указанную ветку или коммит
  • git merge [ветка] - слияние указанной ветки с текущей
  • git log - просмотр истории коммитов
  • git diff - просмотр изменений между коммитами, ветками и т.д.
